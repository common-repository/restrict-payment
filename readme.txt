Restrict selected WooCommerce payment method to old customers with at least one previous completed order.

== Description ==

This powerful plugin allows you to restrict a selected payment method or gateway to old customers only with at least one previous completed order.

=== Features ===

* Restrict a desired payment method or gateway to old customers only.
* Easy automated KYC restriction.
* Custom message displayed to restricted customers.
* Email based WooCommerce payment restriction.
* If the customer used email is associated with at least one completed previous order then the customer will be allowed to use the restricted payment method.
* If the customer used email is not associated with any completed orders then the selected payment gateway will be restricted for that customer.
* Restrict customer payments by email.
* Restrict credit card payments to old customers only!

== Installation ==

* After installing and activating this plugin go to Settings >> Restrict Payment Methods
* Select from the drop-down menu the desired payment gateway you want to restrict to old customers only with at least one previous completed order.
* Insert a desired error message to be displayed to new customers who try to use the restricted payment method.

= Minimum Requirements =

* WordPress 3.8 or greater
* PHP version 5.2.4 or greater

== Frequently Asked Questions ==

= How to whitelist a customer for the restricted payment method? =

Simply create a new order with the customer email then mark it as complete from wp-admin.

= Which payment gateway I can restrict using this plugin? =

You can restrict any WooCommerce gateway using this plugin.

= Why to use this plugin? =

This plugin is useful for high risk businesses who want to limit card payments to old or known customers only to avoid chargebacks.
